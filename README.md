# CodeAlpha_Car_Price_Prediction
## Project Overview
This repository contains a **Car Price Prediction** model developed as part of my Data Science Internship at **CodeAlpha**. The goal is to predict the market price of a vehicle based on its technical specifications.

## Model Performance
- **Algorithm:** Linear Regression
- **R-squared Score:** 0.8180 (81.8% Accuracy)
- **Status:** Completed Successfully

## Key Features Used for Prediction
- **Horsepower:** Engine power output
- **Engine Size:** Total volume of cylinders
- **Curb Weight:** Total weight of the vehicle
- **Highway MPG:** Fuel efficiency
